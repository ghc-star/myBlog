# React 入门

这里开始就是正文内容。

## props 是什么

props 用来让父组件给子组件传值。

## state 是什么

state 用来管理组件自己的状态。

## 什么时候拆组件

当一段 UI 有明确职责，而且未来可能复用时，就可以拆出来。
